                His explainition
                Each container item has a few properties that describe its child elements. so for example, a groupItem has
                some array properties like groupItem.pageItems (this is all children of the container, including other
                containers... except for layers), groupItem.compoundPathItems (this is just top level compound paths that
                exist in groupItem), groupItem.pathItems (this is any top level simple path items like polygons), etc.
